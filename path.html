Data visualization description

The purpose of this visualization is to visualize how the runner moves on the path along the time. And it allow the user to add more than one runners and compare their speed and show the how big difference it is among the runners. And for the runners who wants to compare their records to others can also use this visualization to illustrate the difference. This pages contains two blocks, one is marathon map which is the route of Lausanne marathon and the other is the select menu which allows users to filter their desired player and display it on the map. Moreover, the user could select at most five people from arbitrary year, distance, group and person.
